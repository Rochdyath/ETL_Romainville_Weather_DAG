# ETL_Romainville_Weather_DAG
Automatisation du proccessus d'ETL (Extact - Transform - Load) de la météo de Romainville

## Outils utilisés
* AWS
  * EC2
  * S3 
* API : Open weather map
* Apache airflow
* Python
